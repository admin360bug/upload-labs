# 环境要求：

##### 官方建议：phpstudy（小皮面板）

##### 运行环境：PHP7.3.10~PHP5.2.17

##### 中间件采用：Apache，且设置以moudel方式连接

##### PHP需要开启组件：php_gd2,php_exif
